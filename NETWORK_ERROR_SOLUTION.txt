🔍 HOW TO VERIFY BACKEND IS RUNNING

After starting backend, use ANY of these methods:

Method 1: Command Prompt Console
  ├─ Should show: "[nodemon] exec 'node server.js'"
  ├─ Should show: "✅ Server running on port 5000"
  ├─ Should show: "✅ Database connected"
  └─ Should have NO error messages

Method 2: Check Port
  ├─ Open Command Prompt
  ├─ Run: netstat -ano | findstr :5000
  ├─ Result: Should show active connection
  └─ If empty: Backend not running

Method 3: Health Check
  ├─ Open browser: http://localhost:5000/health
  ├─ Should show: {"status":"healthy",...}
  └─ If shows "Cannot GET /health": Backend not running

Method 4: Network Tab
  ├─ Try login
  ├─ Network tab should show: Request completing
  ├─ Status should show: 200 OK (not PENDING)
  └─ If shows PENDING: Backend still not running

🛠️ TROUBLESHOOTING COMMON ISSUES

Issue 1: "Address already in use"
┌─────────────────────────────────────────────────────┐
│ Cause: Port 5000 already taken by another app       │
│                                                     │
│ Fix:                                                │
│  1. Find process: netstat -ano | findstr :5000      │
│  2. Kill it: taskkill /PID <number> /F              │
│  3. Try npm run dev again                           │
└─────────────────────────────────────────────────────┘

Issue 2: "Cannot find module..."
┌─────────────────────────────────────────────────────┐
│ Cause: npm packages not installed                   │
│                                                     │
│ Fix:                                                │
│  1. Go to server folder: cd server                  │
│  2. Install packages: npm install                   │
│  3. Try npm run dev again                           │
└─────────────────────────────────────────────────────┘

Issue 3: "MongoError: connect ECONNREFUSED"
┌─────────────────────────────────────────────────────┐
│ Cause: MongoDB not running                          │
│                                                     │
│ Fix:                                                │
│  1. Start MongoDB service                           │
│  2. Or check connection string in .env              │
│  3. Try npm run dev again                           │
└─────────────────────────────────────────────────────┘

Issue 4: Still showing PENDING after backend started
┌─────────────────────────────────────────────────────┐
│ Cause: Browser cached old request                   │
│                                                     │
│ Fix:                                                │
│  1. Hard refresh: Ctrl+Shift+R (Windows)            │
│  2. Or: Cmd+Shift+R (Mac)                           │
│  3. Or: Clear browser cache                         │
│  4. Close and reopen browser                        │
│  5. Try login again                                 │
└─────────────────────────────────────────────────────┘
